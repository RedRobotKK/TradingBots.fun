## 🎯 **Available Data Sources**

### **1. BINANCE API** ⭐ RECOMMENDED

**Endpoint:** `https://api.binance.com/api/v3`
**WebSocket:** `wss://stream.binance.com:9443/ws`
**Authentication:** ❌ NOT REQUIRED (public data)
**Rate Limit:** 1,200 requests/minute
**Free Tier:** Yes, forever
**Best For:** Real-time prices, historical OHLCV, most reliable

**Available Endpoints:**
- Real-time ticker: `/ticker/24hr?symbol=SOLUSDT`
- Historical candles: `/klines?symbol=SOLUSDT&interval=1h`
- Order book: `/depth?symbol=SOLUSDT`
- Trade history: `/trades?symbol=SOLUSDT`

**Example Symbols:**
- `SOLUSDT` (Solana)
- `BTCUSDT` (Bitcoin)
- `ETHUSDT` (Ethereum)
- `BNBUSDT` (Binance Coin)
- All major trading pairs available

---

### **2. COINGECKO API**

**Endpoint:** `https://api.coingecko.com/api/v3`
**Authentication:** ❌ NOT REQUIRED (free tier)
**Rate Limit:** 50 requests/minute (free)
**Free Tier:** Yes, forever
**Best For:** Backup pricing, many cryptocurrencies, fallback source

**Available Endpoints:**
- Real-time price: `/simple/price?ids=solana&vs_currencies=usd`
- Historical chart: `/coins/solana/market_chart?vs_currency=usd&days=365`
- Market data: `/coins/markets`

**Example Coin IDs:**
- `solana`
- `bitcoin`
- `ethereum`
- `ripple`
- 15,000+ cryptocurrencies supported

---

### **3. KRAKEN API**

**Endpoint:** `https://api.kraken.com/0/public`
**Authentication:** ❌ NOT REQUIRED (public data)
**Rate Limit:** 15 requests/second (900/minute)
**Free Tier:** Yes, forever
**Best For:** Alternative exchange pricing, historical data

**Available Endpoints:**
- Real-time ticker: `/Ticker?pair=XSOLZUSD`
- Historical OHLC: `/OHLC?pair=XSOLZUSD&interval=60`
- Order book: `/Depth?pair=XSOLZUSD`

**Important:** Kraken uses different pair naming:
- `XSOLZUSD` (not SOLUSDT)
- `XXBTZUSD` (not BTCUSDT)
- `XETHZUSD` (not ETHUSDT)

---

### **4. HYPERLIQUID API** ⭐ YOUR TRADING VENUE

**Endpoint:** `https://api.hyperliquid.com/info`
**WebSocket:** `wss://api.hyperliquid.com/ws`
**Authentication:** ❌ NOT REQUIRED (public data)
**Rate Limit:** UNLIMITED (no rate limit)
**Free Tier:** Yes, you use it anyway for trading
**Best For:** Direct venue pricing, most up-to-date prices

**Available Endpoints:**
- Market data: `/info` (public)
- Orderbook: WebSocket subscription
- Trades: Real-time execution data
- Funding rates: Perpetuals data

**Pairs:** Hyperliquid perpetuals (SOL, BTC, ETH, etc.)

---

### **5. POLYGON.IO API**

**Endpoint:** `https://api.polygon.io/v1`
**Authentication:** ✅ REQUIRES API KEY (free key available)
**Rate Limit:** 5 requests/minute (free tier)
**Free Tier:** Yes, but limited
**Best For:** Aggregate market data, optional

**Note:** Requires free API key from https://polygon.io
**API Key:** See `.env.data-feeds` for storage

## 📊 **Rate Limit Analysis**

### **Real-Time Trading Decision Loop**

```
Symbols: 3 (SOL, BTC, ETH)
Decision Frequency: Every 1 second
Calls per cycle: 1 per symbol = 3 calls/sec

Without caching:
  Per minute: 3 × 60 = 180 calls/min
  Binance limit: 1200/min
  Utilization: 15% ✅

With 500ms caching:
  Per minute: ~120 calls/min
  Binance limit: 1200/min
  Utilization: 10% ✅

Result: Well within limits of all APIs
```

### **Historical Data Fetching**

```
For 1-year backtest of 1 hourly candles:

  Batched in 1000: 9 API calls per symbol

Impact: Negligible ✅
```
